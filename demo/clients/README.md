# Demo данные для CPT Clients

## Описание

Этот файл содержит demo данные для создания записей типа `clients` в WordPress.

## Структура данных

- **version**: Версия структуры данных
- **post_type**: Тип записи WordPress (`clients`)
- **description**: Описание данных
- **source_images**: Путь к исходным изображениям
- **items**: Массив объектов клиентов

## Структура элемента клиента

```json
{
  "title": "Название компании",
  "slug": "url-slug",
  "status": "publish",
  "image": "имя-файла.png",
  "image_alt": "Alt текст для изображения",
  "order": 1
}
```

## Изображения

Все изображения находятся в папке `src/assets/img/brands/`:
- c1.png - c11.png (11 изображений)
- z1.png - z8.png (8 изображений)

Всего: 19 изображений

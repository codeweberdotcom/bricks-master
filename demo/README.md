# Система Demo данных

## Описание

Эта папка содержит demo данные для различных типов записей WordPress.

## Структура

```
demo/
├── clients/          # Demo данные для CPT Clients
│   ├── data.json     # JSON файл с данными
│   └── README.md     # Описание структуры
└── README.md         # Этот файл
```

## Использование

Demo данные используются через функции в `functions/demo/` для автоматического создания записей в WordPress.

## Добавление новых типов

Для добавления нового типа записей:
1. Создайте папку с именем типа записи (например, `projects/`)
2. Создайте файл `data.json` со структурой данных
3. Создайте функции импорта в `functions/demo/`

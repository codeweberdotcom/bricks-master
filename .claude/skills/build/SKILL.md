Выполни сборку темы CodeWeber:

**Предварительная проверка:**
Перед запуском убедись что Laragon запущен (MySQL должен работать).
Gulp при старте запускает `php functions/gulp-theme-info.php`, который загружает `wp-load.php` и подключается к БД, чтобы определить активную тему. Без БД сборка упадёт.

1. Запусти команду: `npm run build` из директории `wp-content/themes/codeweber/`
2. Дождись завершения
3. Выведи краткий отчёт:
   - Успешно / ошибки
   - Что скомпилировано (JS, CSS, какие файлы в dist/)
   - В какую тему выведены файлы (дочерняя или родительская)

Если ошибка `WordPress not loaded` — Laragon не запущен.
Если другая ошибка — покажи текст и объясни как исправить.

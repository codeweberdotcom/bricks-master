# Медиа-лицензии

## Назначение

Модуль **Медиа-лицензии** в теме Codeweber даёт владельцу сайта юридическую защиту: он ведёт учёт и привязку лицензий к медиафайлам (изображениям, видео, документам) в WordPress и хранит данные о том, на каких условиях используется каждый файл, кто правообладатель, кто получил лицензию и к какому проекту или элементу она относится.

Использование модуля даёт возможность:

- вести единый реестр лицензий на медиаконтент;
- привязывать каждый медиафайл к конкретной лицензии;
- хранить скан/копию лицензии (PDF) и метаданные по ней;
- при необходимости обосновывать правомерность использования контента.

## Возможности

### Тип записи «Медиа-лицензия»

Для каждой лицензии создаётся запись со следующими полями:

| Поле | Описание |
|------|----------|
| **Название** | Краткое обозначение лицензии (например, «Фотосток — проект X»). |
| **Тип лицензии** | Royalty-free, Rights-managed, Creative Commons и т.п. |
| **Автор-лицензиар** | Таксономия: правообладатель/автор (ведётся отдельный справочник). |
| **Лицензиат** | Лицо или компания, получившие лицензию. |
| **Для элемента** | Конкретный проект, страница, кампания и т.д. |
| **Дата получения** | Когда лицензия/файл получены. |
| **URL элемента** | Ссылка на лицензированный объект в интернете, если есть. |
| **PDF лицензии** | Прикреплённый файл с текстом/сканом лицензии. |

### Связь медиафайлов с лицензией

- В **редактировании вложения** (Медиафайлы → выбранный файл) можно выбрать лицензию из выпадающего списка. Файл связывается с выбранной записью типа «Медиа-лицензия».
- На **странице редактирования лицензии** (добавление/редактирование записи «Медиа-лицензия») в блоке «Прикреплённые медиафайлы»:
  - отображаются все вложения, у которых указана эта лицензия;
  - кнопка **«Выбрать медиафайлы»** открывает медиатеку; выбранные файлы привязываются к текущей лицензии (сразу по AJAX для существующей записи или при сохранении для новой).

Таким образом, связь «файл ↔ лицензия» можно задать и со стороны файла, и со стороны лицензии.

### Авторы-лицензиары

Таксономия **«Авторы-лицензиары»** (Licensor Authors) позволяет не дублировать имена правообладателей: один раз завести автора и назначать его разным лицензиям. Управление — через подпункт в меню «Медиафайлы».

### Списки и колонки

- В списке **всех лицензий** отображаются тип лицензии, автор-лицензиар, наличие PDF и количество прикреплённых файлов.
- В **медиабиблиотеке** для каждого файла можно вывести колонку «Лицензия» с ссылкой на соответствующую запись лицензии.

## Юридическая защита и практическая польза

Модуль сам по себе **не выдаёт и не заменяет лицензию** — он только фиксирует факты и документы, которые вы храните в админке.

### Что он даёт с точки зрения защиты и дисциплины

1. **Документирование**  
   В одной записи собраны: тип лицензии, кто лицензиат, для какого проекта, дата, при необходимости — PDF. При проверке или споре проще показать, что использование было осознанным и привязанным к документу.

2. **Привязка файла к лицензии**  
   Для каждого изображения/видео видно, какая лицензия к нему относится. Это снижает риск «забыть», откуда права и на каких условиях используется файл.

3. **Хранение PDF лицензии**  
   Возможность прикрепить файл лицензии (договор, письмо стока, скрин с условиями) даёт быстрый доступ к первичному документу при необходимости подтвердить права.

4. **Аудит и контроль**  
   Списки лицензий и прикреплённых файлов помогают при внутренней проверке: какие материалы под какой лицензией, нет ли файлов без привязки к лицензии там, где она обязательна.

### Ограничения

- Модуль **не проверяет** содержание лицензии и не даёт юридических консультаций. Ответственность за соблюдение условий лицензии остаётся на владельце сайта.
- Юридическая сила зависит от того, насколько корректно и полно вы заполняете данные и храните реальные документы (в т.ч. PDF).

Рекомендуется: заполнять лицензии при получении контента, привязывать к лицензии все файлы, используемые на сайте, и хранить актуальные PDF там, где этого требует ваша политика по правам на медиа.

## Где находится в админке

- **Медиафайлы** → пункт меню **«Медиа-лицензии»** — список и добавление лицензий.
- **Медиафайлы** → **«Авторы-лицензиары»** — справочник авторов/правообладателей.
- При редактировании любого медиафайла — поле выбора лицензии в блоке полей вложения.

Файлы модуля в теме: `functions/integrations/image-licenses/image-licenses.php`.

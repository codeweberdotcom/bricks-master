# Интеграция DaData в тему Codeweber

Подключена стандартизация адресов через [DaData API](https://dadata.ru/api/clean/address/) (метод clean/address): разбор строки адреса по полям и подстановка в форму WooCommerce (страна, регион, город, улица, дом, квартира, индекс).

## Где взять ключи

1. Зарегистрируйтесь на [dadata.ru](https://dadata.ru/).
2. Подтвердите почту.
3. В личном кабинете DaData откройте раздел «API-ключи».
4. Скопируйте **API-ключ** (Token) и **Секретный ключ** (Secret).

## Настройка в Redux

В админке WordPress: **Внешний вид → Настройки темы** (или раздел, где подключён Redux) → секция **API**.

- **DaData: включить стандартизацию адресов** — включите переключатель.
- **DaData API Token** — вставьте API-ключ из кабинета DaData.
- **DaData Secret (X-Secret)** — вставьте секретный ключ. Он используется только на сервере и не передаётся в браузер.
- **Где показывать кнопку «Проверить адрес»**:
  - **Редактирование адреса (Мой аккаунт)** — кнопка на странице «Редактировать адрес» в личном кабинете.
  - **Оформление заказа (чекаут)** — кнопки в блоках «Адрес оплаты» и «Адрес доставки» на чекауте.

Сохраните настройки.

## Как пользоваться

1. Откройте форму с адресом (например, «Мой аккаунт → Адреса → Редактировать» или чекаут).
2. В поле **Адрес** начните вводить адрес (например: `Москва Сухонская`).
3. **Подсказки появятся автоматически** при вводе (после 2+ символов) или при фокусе на поле, если в нём уже есть текст. Выберите нужный адрес из списка — поля формы (страна, регион, город, улица, дом, квартира, индекс) заполнятся сами.
4. Либо введите адрес одной строкой и нажмите кнопку **«Проверить и заполнить адрес»** — тогда адрес будет стандартизован и разложен по полям.

Стандартизация и подсказки работают в том числе для адресов в **России**. Секретный ключ и API Token хранятся только на сервере и не передаются во фронтенд.

## Ограничения и ошибки

- При **401/403** (неверный ключ или не подтверждена почта) в интерфейсе отображается сообщение «Сервис проверки адреса временно недоступен». Проверьте ключи и статус аккаунта в кабинете DaData.
- При **429** (слишком много запросов) — «Слишком много запросов. Повторите через несколько секунд.» Лимиты DaData: 20 запросов/сек, 60 новых соединений/мин с одного IP.
- Стоимость: [около 20 коп/запрос](https://dadata.ru/pricing/?product=clean).

## Файлы интеграции

- `functions/integrations/dadata/class-codeweber-dadata.php` — класс вызова API clean/address и маппинг полей в WooCommerce.
- `functions/integrations/dadata/dadata-ajax.php` — регистрация AJAX-обработчика `dadata_clean_address` (nonce, санитизация).
- `src/assets/js/dadata-address.js` — кнопка «Проверить адрес», отправка запроса и подстановка полей в форму.
- Скрипт подключается только на страницах «Редактировать адрес» и/или чекаут (согласно выбранным сценариям в Redux).

Сборка: скрипт копируется в `dist/assets/js/dadata-address.js` при запуске `gulp build:dist` (или используется из `src/` при отсутствии файла в dist).

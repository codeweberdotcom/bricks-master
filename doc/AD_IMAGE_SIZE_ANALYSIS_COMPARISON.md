## Анализ соответствия

### 1. Swiper Slider (серия `c`)

**Параметры отображения:**
- Контейнер: 1400px (XXL)
- Padding контейнера: ~40px (20px с каждой стороны)
- Доступная ширина: ~1360px
- Количество слайдов: 7 на XXL
- Padding слайда: `px-5` = 20px × 2 = 40px
- **Ширина области для изображения: ~1360px / 7 - 40px ≈ 154px**

**Реальные размеры изображений:**
- Минимальная ширина: 194px
- Максимальная ширина: 272px
- Средняя ширина: ~235px

**Анализ:**
- ❌ **НЕ СООТВЕТСТВУЕТ**: Изображения (194-272px) **БОЛЬШЕ** области отображения (~154px)
- Изображения будут масштабироваться вниз примерно на **35-45%**
- На Retina дисплеях качество может ухудшиться из-за масштабирования

**Вывод:** Размеры изображений серии `c` **слишком большие** для области отображения в Swiper.

---

### 2. Grid Layout (серия `z`)

**Параметры отображения:**
- Контейнер: 1320px (XL)
- Количество колонок: 4 на XL
- Gap между колонками: `gx-xl-12` = 12px × 3 = 36px
- Padding: `px-xxl-2` = 8px × 2 = 16px
- Доступная ширина: ~1320px - 36px = 1284px
- **Ширина области для изображения: ~1284px / 4 - 16px ≈ 305px**

**Реальные размеры изображений:**
- Ширина: 450px
- Высота: 301px

**Анализ:**
- ❌ **НЕ СООТВЕТСТВУЕТ**: Изображения (450px) **БОЛЬШЕ** области отображения (~305px)
- Изображения будут масштабироваться вниз примерно на **32%**
- На Retina дисплеях качество может ухудшиться

**Вывод:** Размеры изображений серии `z` **слишком большие** для области отображения в Grid.

---

### 3. Grid с карточками (серия `z`)

**Параметры отображения:**
- Контейнер: 1320px (XL)
- Количество колонок: 5 на XL
- Gap: `gx-lg-6` = 24px × 4 = 96px
- Padding карточки: `p-md-8` = 32px × 2 = 64px
- Padding figure: `px-xxl-3` = 12px × 2 = 24px
- Доступная ширина: ~1320px - 96px = 1224px
- **Ширина области для изображения: ~1224px / 5 - 64px - 24px ≈ 160px**

**Реальные размеры изображений:**
- Ширина: 450px
- Высота: 301px

**Анализ:**
- ❌ **НЕ СООТВЕТСТВУЕТ**: Изображения (450px) **БОЛЬШЕ** области отображения (~160px)
- Изображения будут масштабироваться вниз примерно на **64%**
- Значительное ухудшение качества из-за большого масштабирования

**Вывод:** Размеры изображений серии `z` **значительно больше** области отображения в Grid с карточками.

## Итоговый анализ

### Проблемы

1. **Серия `c` (Swiper):**
   - Изображения 194-272px для области ~154px
   - Избыточный размер: **~26-77% больше**
   - Лишний трафик и время загрузки

2. **Серия `z` (Grid):**
   - Изображения 450px для области ~305px
   - Избыточный размер: **~48% больше**
   - Лишний трафик и время загрузки

3. **Серия `z` (Grid с карточками):**
   - Изображения 450px для области ~160px
   - Избыточный размер: **~181% больше**
   - Критическое несоответствие

### Рекомендации

#### Для серии `c` (Swiper)
**Оптимальный размер:** 200x60px (crop: false)
- Ширина: 200px (достаточно для области ~154px + Retina)
- Высота: 60px (соответствует среднему значению 66px)
- Экономия трафика: ~30-40%

#### Для серии `z` (Grid)
**Оптимальный размер:** 300x200px (crop: false)
- Ширина: 300px (достаточно для области ~305px)
- Высота: 200px (пропорционально)
- Экономия трафика: ~33%

#### Универсальный размер (рекомендуется)
**Оптимальный размер:** 300x200px (crop: false)
- Подходит для всех вариантов использования
- Достаточно для Retina дисплеев
- Оптимальный баланс качества и размера

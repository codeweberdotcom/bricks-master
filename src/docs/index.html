Структура файлов

  • dev Все файлы шаблона с неминифицированными CSS / JS. Используйте эту папку, если не хотите использовать Node.js / Gulp / SASS.
  • dist Готовые файлы шаблона (HTML / CSS / JS), собранные из папки src с помощью Node.js / Gulp (компиляция и минификация).
  • src Исходные файлы шаблона (HTML / SCSS / CSS / JS), из которых собирается папка dist.
    • src/assets/js Основной theme.js и сторонние JS. Чтобы отключить плагины, удалите ненужные файлы из src/assets/js/vendor, уберите их вызовы из src/assets/js/theme.js и пересоберите проект.
    • src/assets/css Сторонние CSS. Чтобы отключить плагины, удалите ненужные файлы из src/assets/css/vendor и пересоберите.
    • src/assets/scss SCSS проекта, компилируемые в dist/assets/css.
  • dev/docs и dist/docs — документация шаблона (установка, блоки, элементы и т.д.). Документация также доступна в живой демо-версии.
  • package.json Список зависимостей для установки через npm.
  • gulpfile.js Конфигурация Gulp и описание задач. Подробнее — в официальной документации Gulp.

Установка

Этот шаг для продвинутых пользователей и необязателен. Codeweber можно настраивать обычными HTML / CSS / JS; нужные файлы лежат в папке dev.

Чтобы ускорить разработку с помощью Node.js / Gulp / SASS, выполните:

  1. Установите Node.js, если его ещё нет.
  2. Выполните npm install gulp-cli -g, если Gulp CLI ещё не установлен глобально.
  3. Распакуйте шаблон и в корневой папке проекта codeweber в терминале выполните npm install — установятся зависимости из package.json.
  4. После этого проект готов к работе, все пакеты npm установлены.

Команды Gulp

  • gulp Запускает задачу по умолчанию: создание папки dist, BrowserSync, объединение библиотек, минификация JS, компиляция SASS и HTML, запуск watch.
  • gulp serve Запускает watch и BrowserSync: изменения в HTML / SCSS / CSS / JS автоматически компилируются.
  • gulp build:dist Создаёт папку dist с минифицированными CSS и JS.
  • gulp build:dev Создаёт папку dev с неминифицированными CSS и JS.

Чтобы остановить сервер, нажмите Control + C

Видео по установке

В этом коротком видео показано:

  • Установка пакетов npm (команда: npm install)
  • Запуск gulp (команда: gulp serve)
  • Смена основного цвета.

Accordion with Icon on the Right

Добавьте класс icon-right к элементу .accordion-wrapper, чтобы переместить иконку переключателя на правую сторону.

Use this variation when you want the toggle icon to appear on the right side of the accordion header, matching the theme styles.

All arrow and icon behaviors are handled by the theme SCSS with the icon-right helper class.

Combine this class with dark section backgrounds to keep contrast and readability.

This plain accordion item uses the same dark theme styles without card background.

Use plain items when you want the dark theme without additional card background.

This item stays open even when another item is expanded.

Use this behavior together with the block setting “Allow Multiple Items Open”.

Vertical Menu

Вертикальное меню с раскрывающимися подуровнями (collapse) выводится блоком Menu плагина Codeweber Gutenberg Blocks: выберите режим «WP Menu», ориентацию «Vertical», включите «Use Collapse» и тип списка 1, 2 или 3. Ниже — по одному примеру на каждый тип и варианты темы.

@@include('docs/_sidebar.html', {"page": "elements-vertical-menu"})
On this page
  • Type 1 — compact
  • Type 2 — with borders
  • Type 3 — border and padding
  • Type 4 — simple list
  • Dark theme
  • Dark + container class

Type 1 — compact list

Минимальные отступы, без границ. В блоке Menu выберите Menu Style: 1. Контейнер меню получает класс navbar-vertical menu-collapse-nav, список — navbar-nav list-unstyled menu-collapse-1.

  • Menu item 1
    • Submenu item 1.1
      • Submenu item 1.2
  • Menu item 2
Показать пример кода 
<nav class="navbar-vertical menu-collapse-nav" id="menu-collapse-doc-1">
  <ul class="navbar-nav list-unstyled menu-collapse-1">
    <li class="nav-item parent-collapse-item parent-item collapse-has-children">
      <div class="menu-collapse-row d-flex align-items-center justify-content-between">
        <a href="#" class="nav-link d-block flex-grow-1">Menu item 1</a>
        <button type="button" class="btn-collapse w-5 h-5 collapsed" data-bs-toggle="collapse" data-bs-target="#menu-doc-1-item-51" aria-expanded="false" aria-controls="menu-doc-1-item-51" aria-label="Expand submenu">
          <span class="toggle_block" aria-hidden="true"><i class="uil uil-angle-down sidebar-catalog-icon"></i></span>
        </button>
      </div>
      <div class="collapse" id="menu-doc-1-item-51" data-bs-parent="#menu-collapse-doc-1">
        <ul class="navbar-nav list-unstyled menu-collapse-1">
          <li class="nav-item parent-collapse-item collapse-has-children">
            <div class="menu-collapse-row d-flex align-items-center justify-content-between">
              <a href="#" class="nav-link d-block flex-grow-1">Submenu item 1.1</a>
              <button type="button" class="btn-collapse w-5 h-5 collapsed" data-bs-toggle="collapse" data-bs-target="#menu-doc-1-item-52" aria-expanded="false" aria-controls="menu-doc-1-item-52" aria-label="Expand submenu">
                <span class="toggle_block" aria-hidden="true"><i class="uil uil-angle-down sidebar-catalog-icon"></i></span>
              </button>
            </div>
            <div class="collapse" id="menu-doc-1-item-52" data-bs-parent="#menu-doc-1-item-51">
              <ul class="navbar-nav list-unstyled menu-collapse-1">
                <li class="nav-item parent-collapse-item"><a href="#" class="nav-link d-block">Submenu item 1.2</a></li>
              </ul>
            </div>
          </li>
        </ul>
      </div>
    </li>
    <li class="nav-item parent-collapse-item parent-item"><a href="#" class="nav-link d-block">Menu item 2</a></li>
  </ul>
</nav>
Показать пример кода (codeweber_nav) 
// Таксономия (рубрики), тип списка 1, все уровни
echo codeweber_nav( 'tax', 'category', [ 'list_type' => '1' ] );

// Или записи типа «Страница» (иерархия)
echo codeweber_nav( 'cpt', 'page', [ 'list_type' => '1' ] );
Показать пример кода (wp_nav_menu) 
wp_nav_menu( array(
  'menu'            => 19,
  'depth'           => 0,
  'container'       => 'nav',
  'container_class' => 'navbar-vertical menu-collapse-nav',
  'menu_class'      => 'navbar-nav list-unstyled menu-collapse-1',
  'walker'          => new CodeWeber_Menu_Collapse_Walker(),
  'fallback_cb'     => false,
  'echo'            => true,
) );
Показать пример кода (шорткод) 
[menu_collapse menu="19" list_type="1"]

Type 4 — simple list

Простой список без collapse: корневой ul с классами list-unstyled text-reset menu-list-type-4, вложенные уровни — list-unstyled text-reset menu-type-4-sub (отступ и margin задаются темой). В блоке Menu выберите Menu Style: 4. Подходит для сайдбара и многоуровневых списков ссылок.

  • Get Started
  • Forms
  • FAQ
    • General
    • Technical
  • Changelog
  • Credits
Показать пример кода 
<nav id="collapse-usage">
  <ul class="list-unstyled text-reset menu-list-type-4">
    <li><a href="../docs/index.html" class="active">Get Started</a></li>
    <li><a href="../docs/forms.html">Forms</a></li>
    <li>
      <a href="../docs/faq.html">FAQ</a>
      <ul class="list-unstyled text-reset menu-type-4-sub">
        <li><a href="../docs/faq.html#general">General</a></li>
        <li><a href="../docs/faq.html#technical">Technical</a></li>
      </ul>
    </li>
    <li><a href="../docs/changelog.html">Changelog</a></li>
    <li><a href="../docs/credits.html">Credits</a></li>
  </ul>
</nav>
Показать пример кода (codeweber_nav) 
echo codeweber_nav( 'tax', 'category', [ 'list_type' => '4' ] );
// или
echo codeweber_nav( 'cpt', 'page', [ 'list_type' => '4' ] );
Показать пример кода (wp_nav_menu) 
wp_nav_menu( array(
  'menu'            => 19,
  'depth'           => 2,
  'container'       => 'nav',
  'container_id'    => 'collapse-usage',
  'container_class' => '',
  'menu_class'      => 'list-unstyled text-reset menu-list-type-4',
  'walker'          => new CodeWeber_Menu_Collapse_Walker(),
  'list_type'       => '4',
  'fallback_cb'     => false,
  'echo'            => true,
) );
Показать пример кода (шорткод) 
[menu_collapse menu="19" list_type="4" depth="2"]

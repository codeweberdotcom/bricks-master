Vertical Menu

Вертикальное меню с раскрывающимися подуровнями (collapse) выводится блоком Menu плагина Codeweber Gutenberg Blocks: выберите режим «WP Menu», ориентацию «Vertical», включите «Use Collapse» и тип списка 1, 2 или 3. Ниже — по одному примеру на каждый тип и варианты темы.

@@include('docs/_sidebar.html', {"page": "elements-vertical-menu"})
On this page
  • Type 1 — compact
  • Type 2 — with borders
  • Type 3 — border and padding
  • Type 4 — simple list
  • Type 5 — dropdown right

Type 1 — compact list

Минимальные отступы, без границ. В блоке Menu выберите Menu Style: 1. Контейнер: navbar-vertical menu-collapse-nav navbar-light или navbar-dark, список — navbar-nav list-unstyled menu-collapse-1.

navbar-light

  • Menu item 1
    • Submenu item 1.1
      • Submenu item 1.2
  • Menu item 2

navbar-dark

  • Menu item 1
    • Submenu item 1.1
      • Submenu item 1.2
  • Menu item 2
Показать пример кода 
<!-- Светлая тема -->
<nav class="navbar-vertical menu-collapse-nav navbar-light" id="menu-collapse-doc-1">
  <ul class="navbar-nav list-unstyled menu-collapse-1">
    <li class="nav-item parent-collapse-item parent-item collapse-has-children">
      <div class="menu-collapse-row d-flex align-items-center justify-content-between">
        <a href="#" class="nav-link d-block flex-grow-1">Menu item 1</a>
        <button type="button" class="btn-collapse w-5 h-5 collapsed" data-bs-toggle="collapse" data-bs-target="#menu-doc-1-item-51" aria-expanded="false" aria-controls="menu-doc-1-item-51" aria-label="Expand submenu">
          <span class="toggle_block" aria-hidden="true"><i class="uil uil-angle-down sidebar-catalog-icon"></i></span>
        </button>
      </div>
      <div class="collapse" id="menu-doc-1-item-51" data-bs-parent="#menu-collapse-doc-1">
        <ul class="navbar-nav list-unstyled menu-collapse-1">
          <li class="nav-item parent-collapse-item collapse-has-children">
            <div class="menu-collapse-row d-flex align-items-center justify-content-between">
              <a href="#" class="nav-link d-block flex-grow-1">Submenu item 1.1</a>
              <button type="button" class="btn-collapse w-5 h-5 collapsed" data-bs-toggle="collapse" data-bs-target="#menu-doc-1-item-52" aria-expanded="false" aria-controls="menu-doc-1-item-52" aria-label="Expand submenu">
                <span class="toggle_block" aria-hidden="true"><i class="uil uil-angle-down sidebar-catalog-icon"></i></span>
              </button>
            </div>
            <div class="collapse" id="menu-doc-1-item-52" data-bs-parent="#menu-doc-1-item-51">
              <ul class="navbar-nav list-unstyled menu-collapse-1">
                <li class="nav-item parent-collapse-item"><a href="#" class="nav-link d-block">Submenu item 1.2</a></li>
              </ul>
            </div>
          </li>
        </ul>
      </div>
    </li>
    <li class="nav-item parent-collapse-item parent-item"><a href="#" class="nav-link d-block">Menu item 2</a></li>
  </ul>
</nav>
Показать пример кода (codeweber_nav) 
// Таксономия (рубрики), тип списка 1, все уровни
echo codeweber_nav( 'tax', 'category', [ 'list_type' => '1' ] );

// Или записи типа «Страница» (иерархия)
echo codeweber_nav( 'cpt', 'page', [ 'list_type' => '1' ] );
Показать пример кода (wp_nav_menu) 
wp_nav_menu( array(
  'menu'            => 19,
  'depth'           => 0,
  'container'       => 'nav',
  'container_class' => 'navbar-vertical menu-collapse-nav navbar-light',
  'menu_class'      => 'navbar-nav list-unstyled menu-collapse-1',
  'walker'          => new CodeWeber_Menu_Collapse_Walker(),
  'fallback_cb'     => false,
  'echo'            => true,
) );
// Тёмная тема: theme => 'dark' даёт navbar-dark в container_class
Показать пример кода (шорткод) 
[menu_collapse menu="19" list_type="1"]

Type 4 — simple list

Простой список без collapse: контейнер navbar-vertical navbar-light или navbar-dark, корневой ul — list-unstyled menu-list-type-4, вложенные — list-unstyled menu-type-4-sub. В блоке Menu выберите Menu Style: 4.

navbar-light

Type 5 — dropdown right

Вертикальное меню с выпадающими подменю вправо (Bootstrap dropend). Открытие по наведению, каретка справа. Контейнер: navbar-vertical navbar-vertical-dropdown, список: navbar-nav flex-column. Шорткод: list_type="5".

navbar-light

  • Пункт без подменю
  • Пункт с подменю
    • Подпункт 1
    • Вложенное
      • Подпункт 2.1
  • Ещё пункт

navbar-dark

  • Пункт без подменю
  • Пункт с подменю
    • Подпункт 1
    • Вложенное
      • Подпункт 2.1
  • Ещё пункт
Показать пример кода 
<!-- Светлая тема -->
<nav class="navbar-vertical navbar-vertical-dropdown navbar-light">
  <ul class="navbar-nav flex-column">
    <li class="nav-item parent-item"><a href="#" class="nav-link">Пункт без подменю</a></li>
    <li class="nav-item parent-item dropdown parent-link dropend">
      <a href="#" class="nav-link" id="menu-item-link-1">Пункт с подменю</a>
      <span class="dropdown-toggle" aria-hidden="true"></span>
      <ul class="dropdown-menu rounded-0" aria-labelledby="menu-item-link-1" role="menu">
        <li class="nav-item"><a href="#" class="dropdown-item">Подпункт 1</a></li>
        <li class="nav-item dropdown dropend parent-link dropdown-submenu">
          <a href="#" class="dropdown-item" id="menu-item-link-2">Вложенное</a>
          <span class="dropdown-toggle" aria-hidden="true"></span>
          <ul class="dropdown-menu rounded-0" aria-labelledby="menu-item-link-2" role="menu">
            <li class="nav-item"><a href="#" class="dropdown-item">Подпункт 2.1</a></li>
          </ul>
        </li>
      </ul>
    </li>
    <li class="nav-item parent-item"><a href="#" class="nav-link">Ещё пункт</a></li>
  </ul>
</nav>
Показать пример кода (codeweber_nav) 
echo codeweber_nav( 'tax', 'category', [ 'list_type' => '5' ] );
// или
echo codeweber_nav( 'cpt', 'page', [ 'list_type' => '5' ] );
// с глубиной и темой:
echo codeweber_nav( 'tax', 'category', [ 'list_type' => '5', 'depth' => 3, 'theme' => 'dark' ] );

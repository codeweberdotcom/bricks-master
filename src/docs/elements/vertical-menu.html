Вертикальное меню

Вертикальное меню с раскрывающимися подуровнями (collapse) выводится блоком Menu плагина Codeweber Gutenberg Blocks: выберите режим «WP Menu», ориентацию «Vertical», включите «Use Collapse» и тип списка 1, 2 или 3. Ниже — по одному примеру на каждый тип и варианты темы.

Type 1 — компактный список

Минимальные отступы, без границ. В блоке Menu выберите Collapse list type: 1. Контейнер меню получает класс navbar-vertical menu-collapse-nav, список — navbar-nav list-unstyled menu-collapse-1.

Type 2 — с границами (карточный вид)

Пункты верхнего уровня с рамкой сверху и по бокам. В блоке Menu выберите Collapse list type: 2. Список использует класс menu-collapse-2; стили задают границы для .nav-item.parent-item.

Type 3 — рамка контейнера и отступы

Внешняя рамка у nav, разделители между пунктами верхнего уровня, увеличенные отступы. В блоке Menu выберите Collapse list type: 3. Список — menu-collapse-3; у контейнера при этом типе тема добавляет общую рамку и скругление.

Тёмная тема

Для тёмного фона в блоке Menu выберите Theme: Dark. К контейнеру меню добавляется класс navbar-vertical-dark, к ссылкам и кнопке — text-white. Оберните блок в карточку с card bg-dark для контраста.

Тёмная тема с классом контейнера

Дополнительные классы для контейнера меню задаются в блоке Menu в поле Container class. Например, border px-5 добавляет рамку и горизонтальные отступы к самому nav. Удобно для выделения блока на тёмном фоне.

Input

Показать примеры кода

  <input id="textInputExample" type="text" class="form-control" placeholder="Text Input">
  <label for="textInputExample">Text Input</label>
</div>

</div>

Показать примеры кода

</div>

Small Checks

Используйте класс form-check-sm для уменьшенных чекбоксов и радио-кнопок (font-size: 0.7rem, input: 0.7rem).

Показать примеры кода 
<div class="form-check form-check-sm">
  <input class="form-check-input" type="checkbox" value="" id="flexCheckSmDefault">
  <label class="form-check-label" for="flexCheckSmDefault"> Small checkbox </label>
</div>
<div class="form-check form-check-sm">
  <input class="form-check-input" type="checkbox" value="" id="flexCheckSmChecked" checked>
  <label class="form-check-label" for="flexCheckSmChecked"> Small checked checkbox </label>
</div>

Dark Background

Формы на тёмном фоне. Оберните контейнер в bg-dark и добавьте data-bs-theme="dark" для автоматической инверсии цветов элементов форм.

Показать примеры кода 
<div class="bg-dark p-6 rounded" data-bs-theme="dark">
  <div class="form-floating mb-4">
    <input id="darkTextInput" type="text" class="form-control" placeholder="Text Input">
    <label for="darkTextInput">Text Input</label>
  </div>
  <!-- /.form-floating -->
  <div class="form-floating mb-4">
    <textarea id="darkTextarea" class="form-control" placeholder="Textarea" style="height: 150px"></textarea>
    <label for="darkTextarea">Textarea</label>
  </div>
  <!-- /.form-floating -->
  <div class="form-select-wrapper mb-4">
    <select class="form-select" aria-label="Dark select example">
      <option selected>Open this select menu</option>
      <option value="1">One</option>
      <option value="2">Two</option>
      <option value="3">Three</option>
    </select>
  </div>
  <!-- /.form-select-wrapper -->
  <div class="form-check">
    <input class="form-check-input" type="checkbox" value="" id="darkCheckDefault">
    <label class="form-check-label" for="darkCheckDefault"> Default checkbox </label>
  </div>
  <div class="form-check mb-4">
    <input class="form-check-input" type="checkbox" value="" id="darkCheckChecked" checked>
    <label class="form-check-label" for="darkCheckChecked"> Checked checkbox </label>
  </div>
  <div class="form-check">
    <input class="form-check-input" type="radio" name="darkRadio" id="darkRadio1">
    <label class="form-check-label" for="darkRadio1"> Default radio </label>
  </div>
  <div class="form-check">
    <input class="form-check-input" type="radio" name="darkRadio" id="darkRadio2" checked>
    <label class="form-check-label" for="darkRadio2"> Checked radio </label>
  </div>
</div>
